定义规则
根据规则去查找
JavaScript 中定义正则表达式的语法有两种
1. 定义正则表达式语法
const 变量名=/表达式/      /  / 这是定义表达式的规则，/  /里面的是规则 / / 是正则表达式字面量

判断是否有符合规则的字符串：
test() 方法 用来查看正则表达式与指定的字符串是否匹配
语法：
正则名.test(被检测的字符串)  //如果正则表达式与指定的字符串匹配 ，返回true，否则false

检索（查找）符合规则的字符串：
exec() 方法  在一个指定字符串中执行一个搜索匹配
语法：
正则名.exec(被检测的字符串)   //如果匹配成功，exec() 方法返回一个数组，否则返回null
