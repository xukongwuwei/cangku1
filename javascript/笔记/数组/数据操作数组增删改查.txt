数组本质是数据集合，数据操作无非就是增，删，改，查

查：
查询数组数据   数组[下标]

改：
重新赋值  数组[下标]=新值

增：
数组添加新的数据  数组名.push(新增的内容)     数组名.unshift(新增的内容)
数组名.push(） 方法将一个或多个元素添加到数组的末尾，并返回该数组的新长度，语法：数组名.push(元素1，元素2，...，元素n)
数组名.unshift() 方法将一个或多个元素添加到数组的开头，并返回该数组的新长度，语法：数组名.unshift(元素1，元素2，...，元素n)


删：
删除数组中的数据  数组名.pop( )    数组名.shift( )    数组名.splice(操作的下标，删除的个数)
数组名.pop( ) 方法从数组中删除最后一个元素，并返回该元素的值，每次只能删一个 语法：数组名.pop( )
 数组名.shift( )方法从数组中删除第一个元素，并返回该元素的值，每次只能删一个 语法：数组名.shift( )
 数组名.splice(操作的下标，删除的个数) 删除指定元素 语法:数组名.splice(下标,要删几个)      数组名.splice(从哪里开始（直接数字就行，这个数字代表下标），删除的个数)   如果没有写删除的个数，默认从起始位置删到最后   -----常用于随机抽奖，删除指定商品
