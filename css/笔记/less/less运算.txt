 Less 运算 任何数字、颜色或者变量都可以参与运算。就是Less提供了加（+）、减（-）、乘（*）、除（/）算术运算。
/*Less 里面写*/
@witdh: 10px + 5;
div {
 border: @witdh solid red;
}
/*生成的css*/
div {
 border: 15px solid red;
}
/*Less 甚至还可以这样 */
width: (@width + 5) * 2;

注意：
乘号（*）和除号（/）的写法  运算符中间左右有个空格隔开 1px + 5
 对于两个不同的单位的值之间的运算，运算结果的值取第一个值的单位
 如果两个值之间只有一个值有单位，则运算结果就取该单位
